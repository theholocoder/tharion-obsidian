## Quand utiliser cette compétence ?

Tu dois TOUJOURS utiliser cette compétence dès qu'il faut rédiger une description pour une scène ou pour une session.
Si il manque des informations, ou des éléments à porter à l'attention des PJ, les demander au MJ (utilisateur)
avant d'écrire en posant une question via l'outil approprié. Mais n'invente rien que le MJ ne t'ait pas demandé.

**Attention** : lorsque tu inventes des informations, prends en compte le lore du dossier @Lore/ pour éviter les incohérences.

## Description d'une scène

Sans trop en dire, une description d'une scène doit permettre aux PJ de savoir ce qu'ils peuvent faire en portant à
leur attention les éléments importants de la scène. Inutile de décrire un chandelier en détails si il n'a aucune
importance pour l'histoire. En revanche, une description détaillée d'une bibliothèque pourrait permettre aux PJ de
trouver le passage secret qu'elle cache.

Par exemple, si on décrit une chambre dans laquelle les PJ peuvent trouver des indices dans un tiroir vérouillé du bureau,
de l'or caché dans le matelas du lit et une bibliothèque cachant une pièce secrète, la description pourrait être :

```description
> Vous entrez dans la pièce, qui comporte un lit en bois dont les draps en soie de couleur violette sont faits au carré.
> Une bibliothèque se dresse sur le mur en face de la porte, des livres sont disposés sur les étagères ainsi que quelques
> bibelots. Un beau bureau en bois comportant deux tiroirs est placé sous la fenêtre, quelques parchemins sont posés
> sur le plan de travail de façon désordonnée.
```

## Format des descriptions

Les descriptions doivent être écrites en Markdown dans un élément de type "citation"/"quote" (préfixé par "> "). Le
texte n'est pas formaté. Un dialogue ou un élément écrit doit être en italique entre guillemets.
