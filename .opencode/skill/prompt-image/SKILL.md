## Quand utiliser cette compétence ?

Tu dois TOUJOURS utiliser cette compétence dès qu'il faut rédiger un prompt pour la génération d'une image.

## Prompt pour générer une image

Le format est un texte simple. Vas droit au but, sacrifie la grammaire au profit d'une description courte mais contenant
tous les détails importants à dessiner. Les informations de lore, les noms des personnages et toute autre information
qui ne sera pas affichée ne doivent pas être incluses dans le prompt.
