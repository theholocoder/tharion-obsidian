>[!info]+ Info
> Le mode intermède représente une succession de jours plutôt que de minutes ou de scènes. Généralement ce mode correspond au moment où vous vous trouvez en sûreté dans une ville pour vous remettre de vos aventures ou étudier un artefact que vous avez découvert, par exemple.

## Activités d'intermède

| Activité d'intermède             | Description                                                                                                                                                                                                                                                                                                                                                                                                                 | Page               |
| -------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------ |
| **Long repos**                   | Vous pouvez passer un jour et une nuit à vous reposer afin de regagner une quantité de points de vie égale à votre modificateur de Constitution (minimum 1) multiplié par deux fois votre niveau.                                                                                                                                                                                                                           |                    |
| **Gagner de l'argent**           | L'action "Gagner de l'argent" vous permet de gagner de l'argent à l'aide d'une grande variété de compétences. N'hésitez pas à les utiliser de façon créative, en concertation avec le MJ.                                                                                                                                                                                                                                   | Player Core p. 229 |
| **Réapprentissage**              | Vous permet de modifier les choix que vous avez faits lors de la création de personnage. Vous pouvez réapprendre de dons, des compétences et certaines capacités de classe. Vous ne pouvez pas réapprendre votre ascendance, votre héritage, votre historique, votre classe ni vos valeurs d'attributs. Le réapprentissage demande du temps et parfois de l'argent.                                                         | Player Core p. 441 |
| **Acheter et vendre des objets** | Objets peu courants, rares, uniques.<br>Etant donné qu'il est difficile de trouver un magasin qui accepte ce que vous voulez vendre ou propose certains articles que vous souhaitez acheter, cette tâche prend 1 jour d'intermède (ajustable en fonction du nombre d'objets). Le prix d'un objet indique son coût à l'achat. Vous pouvez le vendre pour la moitié de son prix. Le MJ peut ajuster les prix à sa discrétion. | Player Core p. 441 |
| **Fabriquer des objets**         | Vous pouvez profiter d'un intermède pour utiliser l'action de compétence Artisanat afin de fabriquer de nouveaux objets à partir de matières premières.                                                                                                                                                                                                                                                                     | Heroic Crafting    |
| **S'entraîner dans une compétence** | Vous pouvez passer une semaine à pratiquer une compétence afin d'améliorer votre maîtrise. Vous devez réaliser avec succès 3 jets de cette compétence (ou d'une compétence connexe déterminée par le MJ) contre le DD du palier actuel : non-qualifié → qualifié (DD qualifié), qualifié → expert (DD expert), expert → maître (DD maître). Vous ne pouvez pas améliorer une compétence au-delà du palier maître via cette activité. | GMG p. 241 |
| **Contrefaire**                  | Vous créez de faux documents ou imitez un document. Peut prendre de une journée à une semaine en fonction du document à contrefaire.                                                                                                                                                                                                                                                                                        | Player Core p. 246 |
| **Soigner une maladie**          | Vous prenez le temps nécessaire pour guérir d'une maladie ou pour vous occuper d'une créature malade.                                                                                                                                                                                                                                                                                                                       | Player Core p. 244 |
| **Autres activités**             | Concertez-vous avec le MJ si vous passez votre intermède différemment. Vous pouvez par exemple acquérir des biens immobiliers, gérer une entreprise, intégrer une guilde, entretenir votre réseau de contacts, fonder une famille ou toute autre activité que votre personnage souhaite effectuer entre deux missions.                                                                                                      |                    |

## Activités d'intermède de bastion
Ces activités seront disponible lorsque vous aurez débloqué l'accès à votre "bastion" (base) et viennent s'ajouter à la liste des activités d'intermède disponibles ci-dessus.

| Activité de bastion              | Description                                                                                                                                                                                                                                                                    | Page     |
| -------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | -------- |
| **Gérer la construction**        | Vous gérez un groupe d'artisans et d'ouvriers pendant qu'ils construisent un module de pièce dans votre bastion, afin de potentiellement raccourcir la durée des travaux. Si vous choisissez cette activité, vous devez rester jusqu'à la fin des travaux même en cas d'échec. | HBH p. 6 |
| **Activer une pièce du bastion** | Certaines pièces nécessitent que vous y passiez un certain nombre de jours d'intermède pour fonctionner. Par exemple la bibliothèque permet de passer une semaine d'intermède afin de faire des recherches sur un sujet particulier.                                           |          |
