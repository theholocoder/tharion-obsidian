# S001 — Checklist post-session (Port-Lointain)

## 1) Capturer “l’état de fin” (avant d’oublier)
- [ ] Dans [[S001 - Notes]] : compléter le **Récap** (où ça s’arrête / intention PJ / conséquences / questions MJ)
- [ ] Noter en 5 lignes : **timeline** (ordre des scènes faites, repos, détours)
- [ ] Lister : PNJ marquants + 1 phrase “ce qu’ils veulent maintenant”

## 2) FoundryVTT — journaux & traces de jeu
- [ ] Renommer/annoter les **journal entries** avec les infos réellement apprises (pas juste “vu X”)
- [ ] Si tu utilises les **jauges** : consigner la valeur finale (Ressources / Capturés) quelque part (journal ou note MJ)
- [ ] Marquer les **tokens/actors** “à réutiliser” (favoris/compendium/tag) : PNJ introduits, boss, types de Cisarelles
- [ ] Sauvegarder un “snapshot” : notes GM, scènes, playlists (histoire de pouvoir relancer vite)

## 3) Factions — interactions & retombées
- [ ] Pour chaque interaction liée à une faction : noter **qui**, **quoi**, **dette/faveur**, **promesse faite**
- [ ] Mettre à jour ton suivi “faction 0–10” (ou autre métrique) et ajouter 1 phrase “pourquoi ça bouge”
- [ ] Lister 1–2 **conséquences** pour la prochaine fois (accès, soutien, suspicion, demande officielle)

## 4) Rumeurs — statut, session, journal interne
Pour chaque rumeur évoquée ou devenue pertinente :
- [ ] `revelee` → true si donnée aux joueurs
- [ ] `sessions` → ajouter [[S001 - Port-Lointain]]
- [ ] Remplir la table **Journal** (session/date/notes)
- [ ] Si les PJ ont “résolu” ou visité : `exploree` → true + note de conclusion

### Rappels S001 (si déclenchés en jeu)
- [ ] [[L'enfant Arraché]] : si Jon est enlevé (Rue Gelevrod) ou si l’idée d’enlèvements est sortie en jeu
- [ ] [[Le Marché Secret]] : si l’entrepôt révèle la caisse Ligue de Veyra + illusion / témoignage

## 5) POI / Hexploration — marquer sur carte
- [ ] Pour chaque rumeur donnée : noter l’**hex** associé et créer/mettre à jour un POI côté Foundry (carte/notes)
- [ ] Ajouter un libellé simple par POI : “accroche + risque + récompense”
- [ ] Si les PJ ont choisi une destination : marquer “**cible prochaine session**” sur l’hex

## 6) Obsidian — nettoyage & préparation de la suite
- [ ] [[S001 - Port-Lointain]] : remplir `resume` (2–5 phrases) + `termine: true` si c’est le cas
- [ ] `Campagnes/EFG/PJ/*` : vérifier que l’historique de session remonte (et ajouter 1 note manuelle si tu fais ça)
- [ ] Si un PNJ/lieu/rumeur “passe de figurant à important” : créer une note dédiée (ou enrichir l’existante) pendant que c’est frais
- [ ] Ajouter 3 bullets “prépa prochaine” : 1 scène d’ouverture probable, 1 menace, 1 récompense/accroche
