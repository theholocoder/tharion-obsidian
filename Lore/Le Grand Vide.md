# Le Grand Vide

## Description

Nom de l'océan en [[Nyssor]] de l'autre côté de [[La Couronne Blanche]].

Il est appelé comme ça parce que beacoup pensaient basculer dans le vide en passant de l'autre côté de la barrière de glace.
